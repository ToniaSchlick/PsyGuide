Technologies

Setting Up

Testing/debugging

Getting up and running is really simple.

Install Prerequisites
  1. Python3
  2. django
  3. django-crispy-forms
    pip install django-crispy-forms
Clone and Run
  1. $ git clone https://github.com/friday-the-13th/Front-end
  2. $ cd ./Front-end/PsyGuideSite
  3. $ python manage.py runserver
  4. Navigate to localhost:8000 in your browser to see the test server.

Server-Side

This guide was used primarily to link all the moving parts together.
uWSGI is used in conjunction with Nginx to serve webpages rendered by django after being passed through.
