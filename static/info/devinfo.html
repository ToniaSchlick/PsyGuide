Setting Up

Testing/debugging

Getting up and running is really simple.

Install Prerequisites
  1. Python3
  2. django
Clone and Run
  1. $ git clone https://github.com/friday-the-13th/Front-end
  2. $ cd ./Front-end/PsyGuideSite
  3. $ python3 manage.py runserver
  4. Navigate to localhost:8000 in your browser to see the test server.

Python control logic is in Front-end/PsyGuideSite/prescriber/
HTML files are in Front-end/PsyGuideSite/prescriber/templates/prescriber/
